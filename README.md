# 道草45 Studio

> **Version 1 / アプリ表示 v0.00.1**<br>
> 思考を書き、寄り道を分類し、その過程を映像として残すためのWindows向け制作ツールです。

![道草45 Studio ロゴ](public/app-logo.png)

## 概要

道草45 Studioは、無限キャンバスへ考えを書き出す「思考モード」と、思考の広がりをスタンプで振り返る「分析モード」を備えたElectronアプリです。

プロジェクトは独自の`.m45`形式で保存できます。キャンバスとマイク音声は録画でき、シーク可能なWebMとして保存します。

## Version 1でできること

### 思考モード

- 筆圧対応のペン描画
- 触れた部分だけを消す部分消去
- パン、ズーム、Undo、Redo
- 色とサイズの変更、ユーザー定義プリセット
- 表示・非表示を切り替えられるレイヤー

### キャンバス背景

- 白、黒、紙色
- 無地、ドット、罫線、方眼
- 模様の間隔変更
- パンとズームに追従するワールド座標基準の模様

### 分析モード

- テーマスタンプとユーザー定義スタンプ
- スタンプ名、色、サイズの編集
- スタンプテンプレートの保存と読込
- スタンプ箱のドラッグ移動
- 道草値の表示切替
- スタンプの使用割合を確認する「集計」
- 動画の最後に提示できる「今回のまとめ」
- まとめカードへのペン入力
- まとめカードのPNG出力

道草値は、分類用スタンプのワールド座標全体から、広がり、多方向性、スタンプ数を使って0〜45で算出します。ズーム率やカメラ位置には影響されません。

### 録画とWebM出力

- キャンバスとマイク音声の同時録画
- 720p、1080p、1440p
- 30 fps、60 fps
- 4、8、12、20 Mbps
- 音声入力デバイスの選択
- 音声レベルと録画時間の表示切替
- WebM収録後のシーク情報再構築
- VP8またはVP9映像とOpus音声を含むWebM保存

録画停止後、WebMのシーク情報を整えてから保存ダイアログを表示します。保存完了の表示が出る前にアプリを終了しないでください。

### プロジェクト

- 新規作成、開く、保存、名前を付けて保存
- `.m45`形式へのキャンバス、カメラ、背景、レイヤー、スタンプ設定の保存
- メニューとキーボードショートカットによる操作

## 動作環境

- Windows 11
- Node.js 22系を推奨
- npm
- マイク
- ペンタブレットまたは液晶ペンタブレットを推奨

基準表示は1920×1080です。Windows表示倍率100%および125%を想定しています。

## セットアップ

```powershell
npm install
npm start
```

## テスト

```powershell
npm test
npm exec tsc -- --noEmit
```

## ビルド

アプリをパッケージ化します。

```powershell
npm run package
```

インストーラーと配布用アーカイブを生成します。

```powershell
npm run make
```

## 主なショートカット

| 操作 | ショートカット |
|---|---|
| 新規作成 | `Ctrl+N` |
| 開く | `Ctrl+O` |
| 保存 | `Ctrl+S` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |
| 録画開始・停止 | `R` |
| パン | `Space`を押しながらドラッグ |
| ズーム | マウスホイール |

## プロジェクト構成

```text
assets/                         配布用アイコン
public/
├─ app-logo.png                 アプリ内ロゴ
└─ vendor/EBML.min.js           WebMメタデータ処理
src/
├─ layers/                      レイヤーUI
├─ recording/RecordingManager.ts
├─ review/                      分析、道草値、スタンプ箱
├─ shared/                      プロジェクト形式と共通型
├─ whiteboard/Whiteboard.tsx    キャンバスと主要UI
├─ main.ts                      Electronメインプロセスとファイル保存
└─ preload.ts                   Renderer向けAPI
```
