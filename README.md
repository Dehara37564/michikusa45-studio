# 道草45 Studio

> **Version 0 / 開発中**  
> 道草45の動画制作を目的とした、Windows向け個人用・無限ホワイトボード兼録画ツールです。

## このリポジトリについて

このリポジトリは、今後の開発基準となる **Version 0** です。
現在は個人利用を前提とした試作段階であり、仕様・保存形式・操作方法は予告なく変更されます。

チャンネル「道草45」で、話しながら思考を無限キャンバスへ書き出し、その過程を音声付きで記録するために開発しています。

## 現在できること

- 白背景の無限キャンバス
- ペン入力と筆圧による線幅変化
- 消しゴム
- キャンバスのパンとズーム
- Undo / Redo
- ペン色と線幅の変更
- プロジェクトの新規作成・保存・読込
- 1920 × 1080 / 30 fpsでのキャンバス録画
- マイク音声の同時収録（48 kHz指定）
- WebM録画後のDuration・SeekHead・Cues再構築
- Windows メディア プレイヤーでの再生・シーク互換性改善

## 現在の制限

- **MP4出力は未実装です。** 現在の録画形式はWebMです。
- ペン描画音の自動生成は未実装です。
- 60 fps、4K、縦動画、複数の音声サンプリング周波数には未対応です。
- 背景は白紙のみです。罫線、方眼、ドット等は今後追加予定です。
- 配布用の安定版ではありません。

## 開発目標

Version 0では、まず次の制作フローを安定させます。

1. 無限キャンバスへ自然に描画する
2. 思考を妨げずにパン・ズーム・修正する
3. マイク音声とキャンバスを安定して録画する
4. 録画結果をMP4として保存する
5. 描画中だけ自然なペン音を生成する

その後、分類・振り返りモード、背景設定、60 fps、各種出力形式などを段階的に追加します。

## 動作環境

- Windows 11
- Node.js 22系を推奨
- npm
- マイク
- ペンタブレットまたは液晶ペンタブレットを推奨

## セットアップ

```powershell
npm install
npm start
```

## ビルド

```powershell
npm run package
```

インストーラー等の配布物を生成する場合は、次を使用します。

```powershell
npm run make
```
